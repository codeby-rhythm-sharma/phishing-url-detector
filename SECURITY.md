# Security Notes

## Threat Model
This tool analyzes URLs to identify common phishing patterns such as suspicious domains, URL shortening, and misleading keywords.

## Limitations
- Heuristic-based detection may produce false positives
- Cannot guarantee detection of newly registered malicious domains
- Should not be used as the sole security control

## Safe Usage
- Use as an additional security layer
- Always verify URLs manually
- Do not rely solely on automated results
